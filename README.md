## 📌 About The Project

A responsive full-stack To-Do List web application that allows users to manage their daily tasks efficiently. Built using Python Flask for the backend and HTML/CSS/JavaScript for the frontend, with full CRUD functionality and a clean, minimal UI.

---

## ✨ Features

- ✅ Add new tasks instantly
- 🗑️ Delete tasks after completion
- 📋 View all pending and completed tasks
- 💾 Persistent data storage
- 📱 Fully responsive — works on mobile & desktop
- ⚡ Fast and lightweight — deployed on Render

## 🚀 Getting Started Locally

### Prerequisites
- Python 3.x installed
- pip package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/Shreyagc24/todo-flask-app.git

# Navigate to project directory
cd todo-flask-app

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```

Then open your browser and go to:
```
http://localhost:5000
```

---
## Create and activate a virtual environment:
    python -m venv venv
### On Windows
venv\Scripts\activate
### On macOS/Linux
source venv/bin/activate

gunicorn app:app
